Solution 2.2.3.x2.1.

Yes, the operation is commutative, since \(|x-y| = |y-x|\text{.}\) The operation is not associative though: \((2\ast 1)\ast 3 = ||2-1| - 3| = |1-3| = 2\) but \(2 \ast (1 \ast 3) = |2- |1-3|| = |2-2| = 0\text{.}\) This is enough to prove that \((\R, \ast)\) is not a group.

Even more, there is not an identity element: while 0 would be the natural choice here, and indeed \(a \ast 0 = |a-0| = a\) is true for all positive \(a\text{,}\) it is not true for negative \(a\text{.}\) Since there is no identity, there can be no inverses.
