The polynomial is not irreducible over \(\bC\) because the only irreducible polynomials with real coefficients over \(\bC\) are the linear polynomials. It should have exactly 7 roots. If we write \(1 = e^0 = e^{2\pi i} = \cdots\) and raise each to the \(1/7\) we get the roots. They are:

\begin{equation*} e^0 = 1, ~~ e^{\frac{2\pi}{7}i}, ~~ e^{\frac{4\pi}{7}i}, ~~ e^{\frac{6\pi}{7}i}, ~~ e^{\frac{8\pi}{7}i}, ~~ e^{\frac{10\pi}{7}i}, ~~ e^{\frac{12\pi}{7}i} \end{equation*}
