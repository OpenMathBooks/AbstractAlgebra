Solution 8.2.2.2.1.

Using the work done in question 1 above, we need \(a = -91\text{.}\)

Why is this? Well we want \((a + \langle 564\rangle)(471 + \langle 564\rangle) = 3+\langle 564\rangle\text{.}\) In other words, we want \(a\cdot 471 + \langle 564\rangle = 3 + \langle 564\rangle\text{,}\) which is the same as asking that \(3 \in a\cdot 471 + \langle 564\rangle\text{.}\) This is another way of requiring

\begin{equation*} 3 = a\cdot 471 + k\cdot 564\text{.} \end{equation*}

We found values of \(a\) and \(k\) to make this so above.
