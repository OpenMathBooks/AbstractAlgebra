For inverses, we much verify that for each \(x \in \R\) there is some other element \(x' \in \R\) such that \(x \ast x' = x' \ast x = 5\) (since \(5\) is the identity). How can \(x \ast x' = 5\text{?}\) Well \(x \ast x' = x + x' - 5 = 5\) means that \(x' = 10 - x\text{.}\) And notice that \((10-x) \ast x = 10 - x + x -5 = 5\) as needed.
