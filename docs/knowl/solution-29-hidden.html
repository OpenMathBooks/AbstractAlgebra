Only one: the trivial homomorphism. There are only two possible kernels: \(\{0\}\) or \(\Z_7\) (since there are only two subgroups of \(\Z_7\)). If \(\{0\}\) is the kernel, then the homomorphism will be injective. But that would mean that the image of the homomorphism would be a 7 elements subgroup of \(\Z_{10}\) which is impossible. Thus the kernel can only be \(\Z_{7}\) which gives the trivial homomorphism (\(\phi(x) = 0\) for all \(x\)).
