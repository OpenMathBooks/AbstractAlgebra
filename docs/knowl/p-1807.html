If we do this for each of the five representations of 1, we get the following five fifth-roots of 1:

\begin{equation*} 1 = e^{i0\pi/5}, ~~ e^{i2\pi/5}, ~~ e^{i4\pi/5}, ~~ e^{i6\pi/5}, ~~ e^{i8\pi/5}\text{.} \end{equation*}

Note that while \(e^{i10\pi/5}\) is also a solution to the equation, this is already listed: \(e^{i10\pi/5} = e^{i2\pi} = e^0 = 1\text{.}\) Similarly, \(e^{i12\pi/5} = e^{i2\pi/5}\text{,}\) and so on.
