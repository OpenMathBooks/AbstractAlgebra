1. We know that \(a\inv b\inv = (ba)\inv\text{.}\) But because \(ab = ba\text{,}\) we get that \((ba)\inv = (ab)\inv = b\inv a\inv\text{.}\) Thus \(a\inv b\inv = b\inv a\inv\text{,}\) which says that \(a\inv\) and \(b\inv\) commute.

  2. Note that \(a = abb\inv\text{.}\) Since \(a\) and \(b\) commute, we get \(a = bab\inv\text{.}\) Multiply by \(b\inv\) on the left: \(b\inv a = a b\inv\text{,}\) which is to say that \(a\) and \(b\inv\) commute.

  3. We want to show that \(a(ab) = (ab)a\text{.}\) Well, \(a(ab) = a(ba)\) since \(a\) and \(b\) commute, and \(a(ba) = (ab)a\) since the group is associative. Therefore \(a\) and \(ab\) commute.

  4. Suppose \(ab = ba\text{.}\) Now consider \(a^2 b^2 = aabb\text{.}\) Since \(a\) and \(b\) commute, we have

    \begin{align*} a^2b^2 \amp = aabb \\ \amp = abab \\ \amp = baab \\ \amp = baba = b^2a^2 \end{align*}

    Thus \(a^2\) and \(b^2\) commute.

  5. We want to show that \((xax\inv)(xbx\inv) = (xbx\inv)(xax\inv)\text{.}\) Well the left hand side can be simplified:

    \begin{equation*} (xax\inv)(xbx\inv) = xabx\inv \end{equation*}

    and the right hand side can be simplified:

    \begin{equation*} (xbx\inv)(xax\inv) = xbax\inv \end{equation*}

    But since \(a\) and \(b\) commute, \(xabx\inv = xbax\inv\) so \(xax\inv\) and \(xbx\inv\) commute.
