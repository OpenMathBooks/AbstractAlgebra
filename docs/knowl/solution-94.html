Solution 8.2.1.7.1.

\(\Q[x]\) is the ring of polynomials in the variable \(x\) with coefficients from \(\Q\text{.}\) This ring is an integral domain.

An example of an ideal in \(\Q[x]\) is \(\langle x^2 + 1\rangle\text{,}\) the set of all multiples of \(x^2 + 1\text{.}\) So this ideal includes \(\{0, x^2 + 1, x^3 + x, 2x^2 + 2, (x+1)(x^2+1), \ldots\}\text{.}\) Note that every non-zero polynomial in the ideal has degree at least 2.

In fact, every ideal in \(\Q[x]\) will be of the form \(\langle r \rangle\) because \(\Q[x]\) is a principle ideal domain (every ideal is principle).
