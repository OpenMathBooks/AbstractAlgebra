Solution 2.2.3.x5.1.

We will see that it is indeed an abelian group. First, check that \(\ast\) is associative:

\begin{equation*} (x\ast y)\ast z = (x+y-5)\ast z = (x+y-5)+z - 5 = x+y+z - 10\text{.} \end{equation*}
\begin{equation*} x\ast (y \ast z) = x \ast (y+z-5) = x+(y+z-5)-5 = x+y+z - 10\text{.} \end{equation*}

So \((x\ast y)\ast z = x \ast (y \ast z)\) and the operation is associative.

What about the identity? We need to find a real number \(e\) such that \(x \ast e = x\text{.}\) But \(x \ast e = x+e-5\text{,}\) and for this to be \(x\) it must be that \(e = 5\text{.}\) We must also have \(e \ast x = x\) and indeed \(5 \ast x = 5+x-5 = x\text{.}\) Thus there is an identity element, \(e = 5\text{.}\)

For inverses, we much verify that for each \(x \in \R\) there is some other element \(x' \in \R\) such that \(x \ast x' = x' \ast x = 5\) (since \(5\) is the identity). How can \(x \ast x' = 5\text{?}\) Well \(x \ast x' = x + x' - 5 = 5\) means that \(x' = 10 - x\text{.}\) And notice that \((10-x) \ast x = 10 - x + x -5 = 5\) as needed.

Thus \((\R, \ast)\) does form a group. It is also easy to check that \(\ast\) is commutative, so this is in fact an abelian group.
