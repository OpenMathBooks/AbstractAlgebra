Solution 3.2.4.1.1.

Yes, \(H\) is normal in \(G\text{.}\) You can see this by checking left vs right cosets. In fact, you see that there will only be two cosets. If you take any \(a \in H\text{,}\) we have \(aH = H = Ha\text{.}\) What about \(b \notin H\text{?}\) You can quickly check that we will have \(bH = Hb\text{.}\)

The factor group \(G/H\) will have order 2, since \(|G/H| = [G:H] = \frac{|G|}{|H|}\text{.}\) Thus writing it's group table will be very easy. Use \(H\) and \(r_1H\) as the two elements. \(H\) is the identity, and \(r_1Hr_1H = H\text{.}\)
