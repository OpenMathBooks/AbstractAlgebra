Note that we can get every 3-cycle if we have the set \(\{(123), (124), (125)\}\text{.}\) To see this, consider the possible forms of 3-cycles: We can get \((1k2) = (12k)^2\text{.}\) We can get \((1ab) = (12b)(12a)(12a)\text{.}\) Similarly \((2ab) = (12b)(12b)(12a)\text{.}\) And then \((abc) = (12a)(12a)(12c)(12b)(12b)(12a)\text{.}\)

Now to get \(\{(123),(124),(125)\}\) we take conjugates:

\begin{equation*} (12)(34)(123)(123)(12)(34) = (124) \end{equation*}
\begin{equation*} (12)(35)(123)(123)(12)(35) = (125) \end{equation*}
