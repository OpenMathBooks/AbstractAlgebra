Exercise C1.

Consider the ring \(\Z_{5} = \{0, 1, 2, 3, 4\}\text{,}\) where both addition and multiplication are done modulo 5.

  1. Although \(\Z_5\) does not contain negative numbers, explain how \(-1\) and \(-3\) still make sense. Which elements of \(\Z_5\) are these?

  2. What is \((-3)2\text{?}\) Use your answer to the previous part to calculate this. Is this what you would want it to be?

  3. Similarly, while \(\frac{1}{3}\) is not an element of \(\Z_5\text{,}\) this should still make sense. Which element of \(\Z_5\) should \(\frac{1}{3}\) be?

  4. Compute \(\frac{1}{3} + \frac{1}{3}\) and \(\frac{1}{3} + 3\) both the way you want to and using what you found in the previous part. Does this work out?
