No. Being a unit means that there is a multiplicative inverse. That would be a coset \(a + \langle 564 \rangle\) that when multiplied by \(471+\langle 564\rangle\) gives the unity \(1+\langle 564\rangle\text{.}\) We would then have \(471a = 1 + 564k\) for some \(k \in \Z\text{.}\) But that would imply that \(\gcd(471, 564) = 1\text{.}\)
