We will see that it is indeed an abelian group. First, check that \(\ast\) is associative:

\begin{equation*} (x\ast y)\ast z = (x+y-5)\ast z = (x+y-5)+z - 5 = x+y+z - 10\text{.} \end{equation*}
\begin{equation*} x\ast (y \ast z) = x \ast (y+z-5) = x+(y+z-5)-5 = x+y+z - 10\text{.} \end{equation*}

So \((x\ast y)\ast z = x \ast (y \ast z)\) and the operation is associative.
