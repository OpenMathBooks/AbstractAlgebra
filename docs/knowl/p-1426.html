  1. The ideals are \(\{(0,0), (1,0), (2,0)\}\) and \(\{(0,0), (0,1), (0,2)\}\text{.}\) The subring that is not an ideal is \(B = \{(0,0), (1,1), (2,2)\}\text{.}\) To see this is not an ideal, note that \((0,1)\cdot (1,1) = (0,1) \notin B\text{,}\) so \(B\) does not absorb products.

  2. Consider two cosets \((0,0) + B = B\) and \((1,0) + B\) (the other coset would be \((0,1) + B\)). There is no problem with defining these cosets, and in fact \(R/B\) will be a quotient group (since \(B\) is a normal subgroup of \(\Z_3\times \Z_3\)).

    Thus the problem must be with multiplication. We would have \(((0,0)+B)((1,0)+B) = (0,0)+B\text{.}\) But another name for \((0,0) + B\) is \((1,1)+B\text{,}\) and \(((1,1)+B)((1,0)+B) = (1,0)+B \ne B\text{.}\) So coset multiplication is not well defined.
