For reflexivity, note that \(x^2 + y^2 = x^2 + y^2\) so \((x,y) \sim (x,y)\text{.}\) For symmetry, note that if \(x^2 + y^2 = a^2 + b^2\) then \(a^2 + b^2 = x^2 + y^2\) (since equality is symmetric). For transitivity, if \(x^2 + y^2 = a^2 + b^2\) and \(a^2 + b^2 = c^2 + d^2\text{,}\) then \(x^2 + y^2 = c^2 + d^2\text{.}\)
