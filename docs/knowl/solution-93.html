Solution 8.2.1.6.1.

In \(\Z\text{,}\) we have \(\langle 3 \rangle = \{\ldots, -3, 0, 3, 6, 9 \ldots\}\text{,}\) the set of all multiples of 3. This is an ideal. In general, \(\langle r \rangle\) is the set of all multiples of \(r\text{,}\) which is always an ideal. In fact, it is a principle ideal since it is an ideal generated by a single element.
