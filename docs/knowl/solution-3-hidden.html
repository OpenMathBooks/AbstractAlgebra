We must show that every point on the plane number is in one and only one of the sets \(A_r\) and that each \(A_r\) is nonempty. Since \(x^2 + y^2\) is the square of the distance from the origin to \((x,y)\text{,}\) no point is in more than one \(A_r\) (this would mean that the point was both distance \(r^2\) and \(s^2\) from the origin). Moreover, every point is in some \(A_r\text{,}\) in particular, for \(r^2 = x^2 + y^2\text{.}\)

Each \(A_r\) is nonempty since the point \((r,0) \in A_r\text{.}\)

The corresponding equivalence relation is \(\sim\) such that \((x,y) \sim (a,b)\) if and only if \(x^2 + y^2 = a^2 + b^2\text{.}\)
