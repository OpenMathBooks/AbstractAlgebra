The ideals are \(\{(0,0), (1,0), (2,0)\}\) and \(\{(0,0), (0,1), (0,2)\}\text{.}\) The subring that is not an ideal is \(B = \{(0,0), (1,1), (2,2)\}\text{.}\) To see this is not an ideal, note that \((0,1)\cdot (1,1) = (0,1) \notin B\text{,}\) so \(B\) does not absorb products.
