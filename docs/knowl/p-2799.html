For pairs of transpositions of the form \((ab)(bc)\) we can write \((ab)(bc) = (abc)\text{,}\) a single 3-cycle (similarly for a pair that look like \((ab)(ac) = (acb)\text{;}\) this case covers all pairs that contain a repeated element in the two transpositions). The only other non-identity pairs of transpositions look like \((ab)(cd)\text{.}\) But those can be written as \((ab)(cd) = (abc)(bcd)\text{.}\)
