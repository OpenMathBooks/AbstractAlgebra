Let's try \((2x)^2\text{.}\) This is really \((x+x)(x+x)\text{,}\) which we can apply the distributive property to. We get \(x(x+x) + x(x+x) = x^2 + x^2 + x^2 + x^2 = 4x^2\text{.}\) It should be clear this generalizes (although a careful proof would use induction). So \((nx)^2 = n^2x^2\text{.}\) In fact, \((nx)^p = n^px^p\)
