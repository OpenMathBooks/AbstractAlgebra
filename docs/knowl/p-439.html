  1. Assume \(ab = e\text{.}\) Then \(a = b\inv\text{,}\) so \(ba = bb\inv\text{,}\) or \(ba = e\text{,}\) as desired.

  2. Note that since \((ab)c = e\text{,}\) we have that \(ab = c\inv\text{.}\) That means that \(c(ab) = e\) as well. Repeat for \(a(bc) = e\) to get the other equation.
