An integral domain is a commutative ring with unity which contains no zero-divisors. That is, there is no non-zero element \(a\) such that \(a \cdot b = 0\) for some non-zero \(b\text{.}\) Equivalently, we could say that an integral domain is a commutative ring with unity which satisfies the cancelation property: that if \(a \ne 0\) and \(ab = ac\) then \(b = c\text{.}\)

Every field (commutative division ring) is an integral domain, but not every integral domain is a field. For example, \(\Z\) is an integral domain, but not a field.
