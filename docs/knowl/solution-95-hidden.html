A quotient ring is \(R/I\) where \(I\) is an ideal of a ring \(R\text{.}\) This is a set of cosets \(a + I\) where \(a\) is an element of \(R\text{.}\) Each coset is a set of element from \(R\text{;}\) those elements that you get by adding \(a\) to an element of \(I\text{.}\)

The operations are defined by \((a+I) + (b+I) = (a+b)+I\) and \((a+I)(b+I) = ab + I\text{.}\) Equivalently, you can take any element from each coset, operate them together, and take the coset which contains the result.
