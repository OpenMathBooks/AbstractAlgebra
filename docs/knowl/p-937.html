To see what goes wrong with defining multiplication, consider \(r_1Hr_1H = r_1r_1H = r_2H\text{.}\) This is the coset \(r_2H = \{r_2, v\}\text{.}\) But since \(r_1H = d_1H\text{,}\) we should get the same set if we multiply \(d_1Hd_1H = d_1d_1H = r_0H\text{.}\) But of course we don't, since \(r_0H = H = \{r_0, h\}\text{.}\)
