For the sum, it is easiest to think of \(z = a+bi\text{,}\) since now \(a+bi+a-bi = 2a\text{.}\) For the product, write the number in polar form: \(z = re^{i\theta}\) so \(\bar z = re^{-i\theta}\text{.}\) Thus \(z\cdot\bar z = re^{i\theta}re^{-i\theta} = r^2e^{0} = r^2\text{.}\)
