Solution 4.3.3.1.

Define a surjective homomorphism \(\psi:\FR \to \R\) by \(\psi(f) = f(0)\text{.}\) That is, \(\psi\) sends a function to its \(y\)-intercept. It is easy to check this is a homomorphism: \(\psi(f+g) = (f+g)(0) = f(0) + g(0) = \psi(f)+ \psi(g)\text{.}\) Also, it is clearly surjective since every real number \(c\) is the \(y\)-intercept for the constant function \(f(x) = c\text{.}\) Further, the kernel of \(\psi\) is precisely \(B\text{.}\)

Thus, by the Fundamental Homomorphism Theorem, \(\FR/B \cong \R\text{.}\)
