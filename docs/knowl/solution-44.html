Solution 5.3.5.1.

(b) Suppose there was some pair \((a,b) \in \Z^2\) that was a solution to the equation. Apply the natural \(\phi:\Z \to \Z_7\text{:}\)

\begin{equation*} \phi(a^2 - 7b^2 - 24) = \phi(0) \end{equation*}
\begin{equation*} \phi(a)^2 - 0 \phi(b) - 3 = 0 \end{equation*}

But this says that \(3\) is a square in \(\Z_7\text{.}\) Checking each element in \(\Z_7\text{,}\) we find this is not the case, so no such \((a,b)\) can exist.
