Using either the corollary to Gauss's lemma or the rational roots theorem, we know the only possible roots are \(\pm 1\text{,}\) \(\pm 2\text{,}\) \(\pm 3\text{,}\) and \(\pm 6\text{.}\) Checking each of these shows that \(x = -2\text{,}\) \(x = -1\) and \(x = 3\) are roots. After doing long division, we find the polynomial factors as

\begin{equation*} (x-3)(x+1)(x+2)(x^2+1)\text{.} \end{equation*}

This is completely factored over \(\Q\) now. The last quadratic factor is irreducible because if it were to factor, it would factor as linear terms, but it has no roots in \(\Q\text{.}\)
