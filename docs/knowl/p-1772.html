You cannot apply Eisenstein's criterion to the polynomial \(p(x) = x^3 - 3x + 1\text{.}\) However, we can apply it to \(p(x+c)\) for a carefully picked value of \(c\text{.}\)

  1. Prove that for any polynomial \(p(x)\) and any constant \(c\text{,}\) if \(p(x+c)\) is irreducible, then so is \(p(x)\)

  2. For \(p(x) = x^3 - 3x + 1\text{,}\) prove that \(p(x+2)\) is irreducible.
