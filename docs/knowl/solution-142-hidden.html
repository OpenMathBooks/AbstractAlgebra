Yes, \(U_{28} \cong G(2) \times G(3)\text{,}\) although it is not obvious how this isomorphism works. It turns out that we can map \(g = hk\) to the pair \((h,k)\text{.}\) That is, the “obvious” isomorphism is correct.

Now \(U(28)\) is NOT isomorphic to \(\Z_4 \times \Z_3\text{,}\) since that would make \(\Z_4 \cong G(2)\text{,}\) which is false. In fact, \(G(2) \cong \Z_2 \times \Z_2\text{,}\) so we have that \(U(28) \cong \Z_2 \times \Z_2 \times \Z_3\text{.}\) Since \(\Z_2 \times \Z_3 \cong \Z_6\text{,}\) we see that \(U(28) \cong \Z_2 \times \Z_6\text{.}\)
