We write \(x^5 = 1\text{.}\) There are lots of ways to write 1 as a complex number though: \(1 = e^0 = e^{i2\pi} = e^{i4\pi} = e^{i6\pi} = e^{i8\pi}\) and so on. However, we can stop there since we are taking 5th roots.

Now if \(x^5 = e^{i2\pi}\text{,}\) then \(x = (x^{5})^\frac{1}{5} = e^{i2\pi/5}\text{.}\)

On the other hand, \(x^5 = e^{i4\pi}\text{,}\) so this says that \(x = e^{i4\pi/5}\text{.}\) This is a different complex number than \(e^{i2\pi/5}\text{,}\) since it has a different angle.

If we do this for each of the five representations of 1, we get the following five fifth-roots of 1:

\begin{equation*} 1 = e^{i0\pi/5}, ~~ e^{i2\pi/5}, ~~ e^{i4\pi/5}, ~~ e^{i6\pi/5}, ~~ e^{i8\pi/5}\text{.} \end{equation*}

Note that while \(e^{i10\pi/5}\) is also a solution to the equation, this is already listed: \(e^{i10\pi/5} = e^{i2\pi} = e^0 = 1\text{.}\) Similarly, \(e^{i12\pi/5} = e^{i2\pi/5}\text{,}\) and so on.

If we plot these five complex numbers we will get five equally spaced points on the unit circle. Notice that if we raise each to a power we jump around the circle, landing on the point at \((1,0)\) on the fifth power, as expected.
