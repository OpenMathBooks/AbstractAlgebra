Solution 4.3.4.1.

Suppose \(H\) is a homomorphic image of \(D_4\text{.}\) Then \(H \cong D_4/K\) where \(K\) is the kernel of the homomorphism. We know that \(K\) is a normal subgroup of \(D_4\text{,}\) which limits the number of possibilities for \(K\text{.}\) We could have \(K = \{r_0\}\) or \(k = D_4\text{,}\) in which case we find that \(H \cong D_4\) or \(H \cong \{e\}\text{,}\) respetively.

The other normal subgroups of \(D_4\) are the three 4-element subgroups, and \(\{r_0, r_2\}\text{.}\) For any of the 4-element subgroups, \(D_4/K\) would have size \(2\text{,}\) so this would say that \(H \cong \Z_2\) (since that is the only 2-element group).

For \(K = \{r_0, r_2\}\text{,}\) we will have \(|H| = 4\text{,}\) so there are two possibilities: \(\Z_4\) or \(\Z_2 \times \Z_2\text{.}\) Now any element not in this kernel, if you square it, you will get something in the kernel. That is, \((aK)^2 = a^2K = K\) for all \(a \in D_4\text{.}\) Thus \(D_4/K \cong \Z_2\times \Z_2\text{.}\)
