  1. The units are the elements of the ring that have multiplicative inverses. So here, \(1, 3, 5, 7\text{.}\)

  2. \(\Z_8\) is a commutative ring (with unity) but is not a division ring, and thus not a field. It is also not an integral domain because it contains zero divisors, such as \(2\) and \(4\) (since \(2\cdot 4 = 0\)).

  3. \(U(8)\) is a group under multiplication, but is not a field. We would still need addition to work, and this subset is not a subring, since \(U(8)\) is not closed under addition.
