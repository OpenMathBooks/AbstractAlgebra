Solution 8.2.2.1.1.

First we perform the Euclidean algorithm:

\begin{align*} 564 = \amp 1\cdot 471 + 93 \\ 471 = \amp 5\cdot 93 + 6 \\ 93 = \amp 15\cdot 6 + 3 \\ 6 = \amp 2\cdot 3 + 0 \end{align*}

so \(\gcd(471,564) = 3\text{.}\)

Now working backwards through the lines of the Euclidean algorithm to find \(r\) and \(s\text{:}\)

\begin{align*} 3 = \amp 93 - 15\cdot 6\\ 3 = \amp 93 - 15\cdot (471 - 5\cdot 93) = -15\cdot 471 + 76\cdot 93\\ 3 = \amp -15\cdot 471 + 76\cdot (564 - 471) = 76\cdot 564 - 91\cdot 471 \end{align*}

so \(r = -91\) and \(s = 76\text{.}\)
